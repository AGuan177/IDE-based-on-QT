# IDE开发计划（2023/09/02）
## 已办部分：
### 一. 完成基础文本编辑器设计，实现以下基本功能：  
    1.支持键盘/鼠标操作，必要的菜单操作
    2.建立文件，保存/打开/另存/关闭
    3.对字符/串的插入/修改/删除，查找/替换
    4.对文本块的复制、粘贴、剪切
    5.实现对字体大小等的改变

    2.对通过编译的程序可投入运行并给出执行结果
